Dificuldades Enfrentadas no Módulo 06

Inicialmente, o grupo identificou os principais formulários necessários para o funcionamento do aplicativo, com destaque para as telas de login e cadastro. A partir disso, foram definidos os campos obrigatórios e as respectivas regras de validação, garantindo a consistência dos dados inseridos pelos usuários.

A implementação foi realizada utilizando os widgets Form e TextFormField, juntamente com validadores customizados para cada tipo de entrada, como e-mail, senha e nome completo. Também foi utilizado o gerenciamento de foco com FocusNode, permitindo uma navegação mais fluida entre os campos dos formulários.

Durante o envio dos dados, foi implementado um controle de estado para indicar carregamento, utilizando indicadores visuais como o CircularProgressIndicator. Além disso, foram utilizados componentes como SnackBar para fornecer feedback imediato ao usuário, informando sucesso ou erro nas operações realizadas.

A implementação seguiu como base os códigos fornecidos pelo professor, o que contribuiu significativamente para o entendimento da abordagem correta e agilizou o desenvolvimento. Dessa forma, não foram encontradas grandes dificuldades durante a execução do módulo.

Como resultado, os formulários desenvolvidos apresentam validação consistente, feedback visual claro e uma melhor experiência para o usuário. Essa etapa também trouxe uma maior robustez ao projeto, preparando a aplicação para lidar de forma mais segura com a entrada de dados.
